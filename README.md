# Lab6-MD

Proyecto de laboratorio para mineria de datos con modelos KNN (regresion y clasificacion) en un notebook.

## Requisitos
- Python 3.9+ (recomendado)
- pip

## Instalacion de dependencias
Desde la raiz del proyecto:

```bash
pip install -r requirements.txt
```

## Como iniciar
1. Abre el notebook principal en VS Code o Jupyter:
   - `main.ipynb`
2. Ejecuta las celdas en orden.

## Archivos principales
- `main.ipynb`: notebook principal del laboratorio.
- `listings.RData`: datos de entrada.
- `metricas_modelos.json`: metricas de modelos anteriores.
